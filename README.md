# Gerenciador de Estoque Python + MySQL

## 📌 Sobre o Projeto

Sistema de gerenciamento de estoque desenvolvido em Python com integração ao MySQL para controle e organização de componentes eletrônicos.

O projeto foi criado para atender uma necessidade real de organização de estoque, permitindo o cadastro, consulta e gerenciamento de componentes armazenados em gaveteiros.

## 🚀 Tecnologias Utilizadas

* Python
* MySQL
* Tkinter
* JSON

## ⚙️ Funcionalidades

* Cadastro de componentes
* Consulta de itens cadastrados
* Atualização de informações
* Controle de componentes por gaveteiro
* Integração com banco de dados MySQL
* Configuração de múltiplos gaveteiros

## 📂 Estrutura do Projeto

```text
gaveteiro_app.py
config_gaveteiros.json
screenshots/
```

## 🖥️ Como Executar

1. Instale o Python 3.
2. Configure o banco de dados MySQL.
3. Ajuste as configurações no arquivo `config_gaveteiros.json`.
4. Execute:

```bash
python gaveteiro_app.py
```

## 🎯 Objetivo

Automatizar o controle de estoque e facilitar a organização de componentes eletrônicos, reduzindo erros de registro e melhorando a rastreabilidade dos itens.
